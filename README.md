## Long-term decline in above-ground carbon sinks in Brazilian tropical and subtropical forests 

This repository contains the code used to process data, perform analyses, and generate the results presented in the manuscript *Long-term decline in above-ground carbon sinks in Brazilian tropical and subtropical forests*. The scripts reproduce the main statistical analyses, figures, and tables reported in the study, supporting transparency and reproducibility of the findings.

Please refer to the manuscript for full methodological details and interpretation of the results.

In the code the census_level.csv refers to Supplementary Data 3 and census_int_level.csv refers to Supplementary Data 4.
